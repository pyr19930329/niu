# 斗牛游戏 APK 打包指引

## 项目信息
- 应用名称：斗牛游戏
- 包名：com.douniu.game
- 游戏服务器：http://124.223.6.177:3002
- 最低Android版本：5.0（API 21）

---

## 方法一：使用 Android Studio（推荐，本地打包）

### 第一步：安装工具
1. 下载安装 [JDK 17](https://adoptium.net/temurin/releases/?version=17)
2. 下载安装 [Android Studio](https://developer.android.google.cn/studio)

### 第二步：打开项目
1. 打开 Android Studio
2. 点击 `Open` → 选择本文件夹 `dou-niu-apk`
3. 等待 Gradle 同步完成（首次需下载依赖，约5~10分钟）

### 第三步：生成 APK
**调试版（直接安装测试用）：**
- 菜单 `Build` → `Build Bundle(s)/APK(s)` → `Build APK(s)`
- APK 路径：`app/build/outputs/apk/debug/app-debug.apk`

**正式版（需要签名）：**
- 菜单 `Build` → `Generate Signed Bundle/APK` → `APK`
- 按向导创建或选择 Keystore 文件
- APK 路径：`app/build/outputs/apk/release/app-release.apk`

## 方法二：免费在线云打包（无需安装任何工具）

### 使用 Appetize.io / PhoneGap Build / Buildozer

#### 推荐：使用 GitHub Actions 自动打包（完全免费）

1. 在 GitHub 创建一个仓库，把 `dou-niu-apk` 文件夹内容上传
2. 创建文件 `.github/workflows/build.yml`，内容如下：

```yaml
name: Build APK
on: [push]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-java@v3
        with:
          java-version: '17'
          distribution: 'temurin'
      - name: Build APK
        run: |
          chmod +x ./gradlew
          ./gradlew assembleDebug
      - name: Upload APK
        uses: actions/upload-artifact@v3
        with:
          name: douniu-game.apk
          path: app/build/outputs/apk/debug/app-debug.apk
```

3. Push 代码后，在 GitHub `Actions` 标签页下载 APK

## 方法三：命令行打包（已安装JDK但未安装Android Studio）

```bash
# 在项目根目录执行
# Windows
gradlew.bat assembleDebug

# Linux/Mac
./gradlew assembleDebug
```

---

## 安装到手机

1. 手机开启「开发者模式」→「允许安装未知来源应用」
2. 将 APK 传输到手机（微信/QQ/数据线均可）
3. 点击安装即可

---

## 注意事项
- 确保服务器 `http://124.223.6.177:3002` 正常运行
- 手机需连接互联网才能正常游戏
- 如需修改服务器地址，编辑 `MainActivity.java` 第 17 行的 `GAME_URL`
